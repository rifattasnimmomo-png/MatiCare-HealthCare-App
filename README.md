# 🤰 MatriCare - Mobile Healthcare App

A comprehensive mobile healthcare application designed for pregnancy monitoring and maternal health tracking. MatriCare provides expectant mothers with real-time health insights, vital sign monitoring, and AI-powered assistance throughout their pregnancy journey.

## ✨ Features

- **📊 Health Dashboard**: Real-time monitoring of vital signs including heart rate, blood oxygen, temperature, and blood pressure
- **📅 Pregnancy Tracking**: Week-by-week pregnancy progress with personalized insights and milestones
- **🤖 AI Assistant**: Intelligent chatbot for pregnancy-related questions and health concerns
- **📱 Daily Health Logging**: Track daily symptoms, mood, and activities
- **📈 Trends & Analytics**: Visualize health trends over time with interactive charts
- **🚨 Alerts & Notifications**: Important reminders for appointments and health alerts
- **👥 Partner Access**: Share pregnancy journey with family members
- **📑 Reports & Appointments**: Manage medical reports and appointment schedules
- **🔒 Privacy & Security**: Secure data management with privacy controls
- **🌙 Dark Mode**: Eye-friendly dark theme support

## 🛠️ Tech Stack

- **Frontend**: React 18.3.1 with TypeScript
- **Styling**: Tailwind CSS 4.1.12
- **UI Components**: Radix UI primitives
- **Icons**: Lucide React, Material UI Icons
- **Charts**: Recharts
- **Animations**: Motion (Framer Motion)
- **Build Tool**: Vite 6.3.5

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/matricare-app.git
cd matricare-app
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:5173](http://localhost:5173) in your browser

## 🏗️ Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

## 🚀 Deployment

### GitHub Pages

1. Update the `base` in `vite.config.ts` to match your repository name
2. Deploy to GitHub Pages:
```bash
npm run deploy
```

### Manual Deployment

Build the project and deploy the `dist` folder to any static hosting service (Netlify, Vercel, etc.)
